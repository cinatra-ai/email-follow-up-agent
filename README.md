# Email Follow-up Agent

Draft a follow-up sequence for recipients who did not reply to the initial outreach. The agent writes a cadence of follow-up emails, each spaced a configurable number of days after the first send, and hands the sequence to you for review before any of it goes live.

## Capabilities

- Draft a multi-step follow-up sequence for an outreach campaign
- Space each follow-up by a configurable day offset from the initial send
- Keep tone and context consistent with the original outreach
- Pause for human review and approval before the cadence is scheduled
- Return the full follow-up sequence as a single reviewable bundle
